This project for the first project in Data Science Nanodegree. I'm interested in Saudi Market so I choose something related to it.
It's about used cars' ads from kaggle: https://www.kaggle.com/datasets/turkibintalib/saudi-arabia-used-cars-dataset
The libraries used: 
- pandas
- numpy
- seaborn
- matplotlib
The code here: used_cars_in_ksa.ipynb
Summary of the result:
- There's a big percentage of the ads is price negotiatible.
- The year of used car manfactured is start from 1936 to 2021.
- Toyta comes first as the most ads from it and Camry, Land Cruiser come on the top type from this company.
